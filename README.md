# Convolution Implementation in C++ for 1x1 and 3x3 kernels
The repository contains convolution implementation in C++. The python script is used for validation
# Prequisites 
- g++
- python3
- torch
- numpy
# How to run 
To run conv2d
```
    sh run_conv2d.sh
```
To run conv3d
```
    sh run_conv3d.sh
```
To run conv3x3x64x128
```
    sh run_conv3x3x64x128.sh
```
